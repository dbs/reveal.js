Q: Where does the library fit in?

Gathering information, obviously. Evaluating the authority of sources, but also crunching data. Communicating via visualizations (for example, GIS).

We can create live, interactive maps using the sources of open data that are increasingly available from municipal, provincial, and federal governments.
